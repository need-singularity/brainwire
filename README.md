## `brainwire/` Python Package

### 구조

```
brainwire/
├── __init__.py              # v2.0.0
├── variables.py             # 12변수 상수, 카테고리, 텐션 가중치
├── profiles/
│   ├── base.py              # StateProfile, Envelope, SafetyOverrides
│   ├── thc.yaml             # THC 25% 프로파일
│   ├── lsd.yaml             # LSD 100μg
│   ├── psilocybin.yaml      # Psilocybin 25mg
│   ├── dmt.yaml             # DMT 30mg (breakthrough)
│   ├── mdma.yaml            # MDMA 125mg
│   └── flow.yaml            # Flow State (endogenous)
├── hardware/
│   ├── devices.py           # 11개 디바이스 레지스트리
│   ├── hal.py               # Hardware Abstraction Layer (핫플러그)
│   ├── safety.py            # 4-레이어 안전 엔진
│   └── configs.py           # Tier 1-4 프리셋
├── engine/
│   ├── transfer.py          # 범용 전달 함수 (12var × N devices)
│   ├── tension.py           # PureField 텐션 계산
│   ├── pid.py               # PID 컨트롤러 뱅크 (12 독립)
│   └── interpolation.py     # 상태 보간/블렌딩/엔벨로프
├── optimizer.py             # 프로파일별 좌표 하강 최적화
├── simulator.py             # 시간축 세션 시뮬레이터 (PID + 호흡)
├── eeg_feedback.py          # G=D×P/I EEG 피드백 모듈
├── bench.py                 # 멀티-스테이트 벤치마크 CLI
└── calc.py                  # 확장 계산기 (민감도, 갭, 블렌딩)
```

### Quick Start

```bash
# 멀티-스테이트 벤치마크
python -m brainwire.bench bench thc --tier 4
python -m brainwire.bench compare thc lsd dmt --tier 4
python -m brainwire.bench tiers thc

# 세션 시뮬레이션
python -m brainwire.simulator thc --tier 4 --duration 600
python -m brainwire.simulator --all --tier 4
python -m brainwire.simulator thc --tier 3 --concentration

# 프로파일 최적화
python -m brainwire.optimizer
python -m brainwire.optimizer --state thc --tier 4

# G=D×P/I 분석
python -m brainwire.eeg_feedback

# 확장 계산기
python -m brainwire.calc sensitivity thc --tier 3
python -m brainwire.calc gap dmt --tier 3
python -m brainwire.calc blend --states thc flow --weights 0.7 0.3

# TECS-L 스타일 가설 벤치마크
python bench_hypotheses.py

# 레거시 벤치마크
python bench_thc_vars.py
python bench_thc_vars.py --levels
python calc.py sensitivity
python calc.py gap
```
